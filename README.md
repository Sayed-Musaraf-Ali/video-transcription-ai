# 🎥 Video Transcription AI

## 📌 Project Description
This project converts the speech from a video into text using OpenAI Whisper and extracts frame captions using Microsoft GIT.

The project was developed in Google Colab using Python.

---

## 🚀 Features
- Upload any video
- Extract speech from the video
- Convert speech into text (Transcription)
- Extract frame captions from video timestamps
- Easy to use Gradio UI
- AI-powered transcription

---

## 🛠 Technologies Used
- Python
- Google Colab
- OpenAI Whisper
- Microsoft GIT-base
- Gradio
- FFmpeg
- OpenCV

---

## 📂 Project Files
- AI_vedioToText.ipynb
- requirements.txt
- README.md

---

## ▶️ How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries using `requirements.txt`.
3. Run all cells.
4. Upload your video.
5. Get the transcription and frame captions.
